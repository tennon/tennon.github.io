学习笔记——《head first python》

发表于 2017-07-28

作为一个准备走互联网路线的大三在校生，在暑假的这段时间给自己下了个目标。要初步掌握一门编程语言。在经过百度、知乎和前辈人士的咨询后，选择了python这门编程语言。
Python是一种编程语言，诞生于20世纪90年代初，自从2004年以后Python的使用率呈线性增长。2011年1月，它被TIOBE编程语言排行榜评为2010年度语言。
作为一门编程语言，Python同样有语句、表达式、操作符、函数、模块、方法和类等等。但它与众不同的地方是：优雅、明确、简单，在这些设计哲学之上Python提供了大量的、功能强大的编程库，并且都是开源的！
所以对于初入门的新手来说，python是一门很适合学习的语言。

在教程的选择上，本人购买了《python基础教程》

和Head First Python

这两本纸质材料、网络版资料看python教程-廖雪峰。

其中《python基础教程》和“python教程-廖雪峰”的这两门教程理论部分比较多，初学者会有一些地方无法理解，学起来比较枯燥。
相对它们来说“Head First Python（中文版）”图书 “黏住”了我。它以特殊的方式排版，由大量的图片和有趣的内容组合构成，帮助读者达到非疲劳的沉浸式学习效果。
